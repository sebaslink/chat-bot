## INSTRUCCIONES DE USO:
1. Ejecutar EJECUTAR_CHATBOT_FINAL.bat
2. El sistema se abrirá automáticamente en tu navegador

## GESTIÓN DE USUARIOS:
1. Como administrador, ve a la pestaña "Usuarios"
2. Crea usuarios con diferentes roles:
   - Asesor: Solo acceso al Chat Masivo
   - Administrador: Acceso completo al sistema
   - Programador: Acceso completo al sistema

## USO DEL SISTEMA:

4. Usa la pestaña "Chat" para conversar con la IA
5. Gestiona la base de datos en la pestaña "Base de Datos"
6. Accede al Chat Masivo desde el botón correspondiente
